This content animates to its natural height using interpolate-size: allow-keywords

No more JavaScript height calculations!

CSS can now smoothly animate to height: auto

/* Enable interpolation to auto */
html {
  interpolate-size: allow-keywords;
}

  }

Dialogs & the Problem with Z-Index

❌ The Old Way: Z-Index Hell

z-index: 9999
z-index: 99999
z-index: 999999
/* Z-index arms race */
.modal { z-index: 9999; }
.tooltip { z-index: 99999; }
.dropdown { z-index: 999999; }

/* Still gets clipped by parent stacking context! */

✅ The New Way: Top Layer

I'm in the top layer!

No z-index needed. I always appear above everything. 

<dialog id="myDialog">
  <h2>Dialog Content</h2>
  <button onclick="myDialog.close()">Close</button>
</dialog>

<script>
myDialog.showModal(); // Moves to top layer
</script>

Variable Size Typography

Fluid Typography with clamp()

Responsive Text

This text scales smoothly between minimum and maximum sizes based on viewport width. 

h1 {
  font-size: clamp(2rem, 5vw, 4rem);
  /* min: 2rem, preferred: 5vw, max: 4rem */
}

p {
  font-size: clamp(1rem, 2.5vw, 1.25rem);
  line-height: clamp(1.4, 1.5, 1.6);
}

Container Query Typography

Container-Based Text

This text adjusts based on its container size, not the viewport. 

.text-container {
  container-type: inline-size;
}

@container (min-width: 300px) {
  h3 { font-size: 1.5rem; }
}

@container (min-width: 500px) {
  h3 { font-size: 2rem; }
}
